# Personal Portfolio & Blog

This is a personal portfolio website and blog built with Astro. It's designed to be a clean, modern, and responsive space to showcase my work and share my thoughts.

## Live Demo

The live version of this project is hosted on GitHub Pages: [https://leeCS-404.github.io](https://leeCS-404.github.io)

## About This Website

This website serves as a central hub for my professional and personal projects. It's a place where I can share my work with the world, write about topics that interest me, and experiment with new web technologies.

### Purpose

The main goal of this website is to provide a comprehensive overview of my skills and experience. It's designed to be a living document that evolves as I grow as a developer and a person.

### Features

- **Responsive Design**: Looks great on all devices, from mobile to desktop.
- **Blog System**: A space for me to share my thoughts on technology, programming, and other topics.
- **Featured Work Section**: A curated collection of my best projects.
- **Dark Mode**: For a more comfortable viewing experience in low-light environments.

## Tech Stack

This website is built with a modern tech stack that allows for a fast, efficient, and enjoyable user experience.

- [Astro](https://astro.build/): A web framework for building fast, content-focused websites.
- [Svelte](https://svelte.dev/): A radical new approach to building user interfaces.
- [MDX](https://mdxjs.com/): An authorable format that lets you seamlessly write JSX in your Markdown documents.
- [TypeScript](https://www.typescriptlang.org/): A typed superset of JavaScript that compiles to plain JavaScript.
